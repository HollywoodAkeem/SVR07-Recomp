# SVR07-Recomp
Using rexglue. Programs runs


How to use:
Rip your opy of svr 2007 and place it in a folder named "assets"
I recomend having a USA/Canada disk. If you have Region Free or Something else, im not sure if it will work. 

Download the exe provided in releases
place exe file in parent folder to assets. 

you should have a directory like this:
.
└── /svr07/
    ├── svr07.exe
    └── /assets/
        └── [wwe svr07 files]
